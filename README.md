# Local RAG Practice

Ein kleines lokales RAG-Projekt mit Python, Chroma, HuggingFace Embeddings und Ollama.

## Features

- PDF-Dokumente laden
- Dokumente in Chunks teilen
- Embeddings erzeugen
- Chroma als Vektordatenbank nutzen
- Lokale Antwortgenerierung mit Ollama

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
